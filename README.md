# Pergola Ukraine

Сайт-візитка компанії Pergola Ukraine — автоматизовані системи для терас.

## Сторінки
- `index.html` — головна
- `avtomatyzovana-pergola.html` — Автоматизована пергола
- `bioklimatyka-pro.html` — Біокліматика Pro
- `zip-systema.html` — ZIP-система

## Публікація на GitHub Pages
1. Створіть репозиторій і завантажте всі файли.
2. Settings → Pages → Source: `main` / root.
3. Сайт буде доступний за адресою `https://<username>.github.io/<repo>/`.

Медіа (відео, фото) розміщені на Cloudinary.
